# Top MNC Companies - DSA Focus Areas & Interview Patterns

## 1. GOOGLE
### Focus Areas (Interview Weight)
- **Arrays & Strings (70%)**: Sliding window, two pointers, prefix sums, substring matching
- **Trees & Graphs (65%)**: BFS/DFS, tree traversals, shortest paths, connectivity
- **Dynamic Programming (60%)**: Optimization problems, state transitions, memoization
- **Hash Tables (55%)**: Efficient lookups, frequency counting, cache problems
- **System Design**: When applicable for L4+ roles

### Interview Format
- **Rounds**: Usually 4-5 coding rounds over 4 weeks
- **Duration**: 45-60 minutes per round
- **Focus**: Optimization, edge cases, follow-up questions
- **Difficulty**: Mix of Medium and Hard LeetCode problems
- **Hiring Bar**: Problem-solving approach > final code quality

### Top DSA Topics
1. Two Pointers & Sliding Window (Container With Most Water, Longest Substring)
2. BFS/DFS (Number of Islands, Pacific Atlantic Water Flow)
3. Dynamic Programming (Longest Increasing Subsequence, Coin Change)
4. Tree Problems (Lowest Common Ancestor, Serialize/Deserialize)
5. Graph Algorithms (Course Schedule, Word Ladder)

### Company-Specific Hints
- Expect follow-up questions asking for optimization
- Write clean, readable code
- Discuss time/space complexity upfront
- Edge cases are heavily tested

### Top DSA Topics
1. Array Rotation & Manipulation (Rotate Array, Majority Element)
2. Linked List Problems (Reverse Linked List, Merge K Sorted Lists)
3. Tree Traversals (Invert Tree, Validate Tree)
4. OOP Design (LRU Cache, File System)
5. Graph Connectivity (Number of Connected Components, Graph Valid Tree)

### Company-Specific Hints
- They value clean, modular code
- Discuss trade-offs explicitly
- Practice explaining complex algorithms simply
- System design becoming more important

## Study Strategy by Target Company

### For Google/Meta (LeetCode-Heavy)
- Focus: Array, String, Tree, Graph, DP
- Questions: 200-300 problems
- Roadmap: 12 weeks full focus
- Mock Interviews: 5-10 full mocks

### For Amazon (Deep Dive Focused)
- Focus: All basics + behavioral practice
- Questions: 150-200 problems + system design
- Roadmap: 10 weeks coding + 2 weeks system design
- Mock Interviews: 3-5 full mocks + behavioral

### For Apple/Microsoft (Optimization Focused)
- Focus: Efficiency, Memory, Optimization
- Questions: 150-200 problems with optimization variants
- Roadmap: 10 weeks with hw-aware focus
- Mock Interviews: 3-5 mocks with optimization emphasis

### For Uber/Netflix (Real-World Focused)
- Focus: Graphs, Caching, System Design
- Questions: 100-150 coding + more design
- Roadmap: 8 weeks coding + 4 weeks design
- Mock Interviews: 2-3 coding + 2-3 system design

### For Goldman Sachs (Math-Heavy)
- Focus: DP, Algorithms, Mathematical thinking
- Questions: 200+ problems with math focus
- Roadmap: 12 weeks with daily math practice
- Mock Interviews: 5-10 full technical mocks
